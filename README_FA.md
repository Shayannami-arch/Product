# ProductApp Single File - GitHub Ready

این نسخه برای راحتی ساخته شده است.
همه منطق برنامه داخل `main.py` است و دیگر به پوشه‌های `model`، `cotroller`، `view` و `tool` نیاز ندارد.

## روش استفاده در GitHub

1. فایل ZIP را Extract کن.
2. همه فایل‌ها و پوشه `.github` را داخل Repository آپلود کن.
3. مطمئن شو فایل `buildozer.spec` در صفحه اصلی Repository دیده می‌شود.
4. برو تب Actions.
5. Workflow با اسم `Build Android APK` را Run کن.
6. بعد از پایان Build، APK از قسمت Artifacts قابل دانلود است.
